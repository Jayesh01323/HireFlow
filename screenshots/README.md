## Screenshot Guidelines

### Technical Specifications

- **Format**: PNG (preferred) or JPG
- **Resolution**: 1920x1080 (minimum)
- **File Size**: Under 2MB per image
- **Quality**: High quality, no compression artifacts
- **Background**: Clean, no personal data visible

### Content Guidelines

- **No Personal Data**: Remove or blur personal information
- **Real Data**: Use realistic sample data
- **Professional Appearance**: Clean, organized interface
- **Consistent Styling**: Use same theme across all screenshots
- **Clear Labels**: Ensure text is readable

### Naming Conventions

Follow the naming conventions in `naming-conventions.md`:
- Use kebab-case
- Include page/feature name
- Include sequence number
- Descriptive suffix

Example: `job-matcher-01-match-analysis.png`

## Tools for Screenshots

### Recommended Tools

- **Windows**: Snipping Tool, ShareX
- **macOS**: Built-in screenshot, CleanShot X
- **Linux**: Flameshot, Shutter
- **Browser**: DevTools screenshot, Full Page Screen Capture

### Browser Extensions

- **Full Page Screen Capture** (Chrome/Firefox)
- **Awesome Screenshot** (Chrome/Firefox)
- **GoFullPage** (Chrome)

## Image Optimization

### Compression

Use tools like:
- TinyPNG (https://tinypng.com/)
- ImageOptim (macOS)
- FileOptimizer (Windows)

### Format Conversion

- PNG for screenshots with text
- JPG for photographs
- WebP for web optimization
